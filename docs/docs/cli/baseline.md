usage: pum baseline [-h] -b BASELINE [--create-table] [--allow-multiple-modules]
### options:
- `-h, --help`: show this help message and exit
- `-b BASELINE, --baseline BASELINE`: Set baseline in the format x.x.x
- `--create-table`: Create the pum_migrations table if it does not exist
- `--allow-multiple-modules`: Allow multiple PUM modules (with separate migration tables) in the same database
